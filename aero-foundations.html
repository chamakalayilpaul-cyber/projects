Why evaluate Shadcn/ui?
* This is Linear's product UI. Sierra AI is extremely private — Linear likely follows a similar shadcn/ui-adjacent design approach.

Sierra AI is extremely private and there are no public references of their product UI. From a video we could reference for product UI, we could see — Inter typeface, monospace for code identifiers, a green brand accent, ~8px corner radius, hairline borders, and a three-panel IDE-style layout. Combined with their confirmed tech stack of React + Next.js + Tailwind CSS + React Aria, this points strongly to a shadcn/ui-adjacent approach — not necessarily shadcn verbatim, but built on the same philosophy: Tailwind for spacing, CSS variables for theming, a headless accessibility primitive library for components. The visual output is indistinguishable from a well-themed shadcn/ui implementation.

Summary
The single biggest difference: sharpness vs softness
Aero reads as sharper and more clinical. shadcn reads as softer and more approachable. That one difference cascades through almost every visual detail.
Corner radius
Aero uses 4px as its web default, shadcn uses 8px. Small number, noticeable difference — Aero components feel crisp and squared-off, shadcn feels softer and rounder. Across a full UI the cumulative effect is significant: Aero reads as traditional enterprise, shadcn reads as modern startup.
Data density
Both systems are comfortable at medium density but handle it differently. Aero's table rows are 56px, inputs 36px — designed for users scanning large amounts of enterprise data. shadcn defaults are leaner at 48px rows and 32px inputs. The more meaningful difference is that shadcn has a built-in convention for tabular monospace numerics — numbers in tables render in a fixed-width font so columns of figures align perfectly. Aero has no equivalent. For a product where users scan columns of ratings, counts, or revenue figures, that alignment makes data significantly easier to read at a glance. It's the same reason Stripe's dashboard feels information-dense yet effortlessly readable.

Other modern products sharing the same aesthetic: Linear, Clerk, Resend, Vercel v0, Railway, Neon, Trigger.dev, Cal.com, Supabase (newer surfaces), Liveblocks, Dub.co, Stripe.
